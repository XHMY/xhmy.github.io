 HOW TO ADD A PAPER
 ------------------
 Copy one block below and edit it. Each paper starts with "## " + its title,
 followed by these fields (the title is required; the rest are optional):

   - Authors: Comma-separated list. Wrap YOUR OWN name in **double asterisks**
              to highlight it, e.g. **Yifan Zeng**. Keep the "✧" mark after a
              name to denote equal contribution.
   - Venue:   Conference / journal / workshop, e.g. "EMNLP 2025".
   - Info:    Extra info shown after the venue (arXiv id, volume, date...).
              Separate multiple pieces with " | " — each renders after a "•".
              e.g.  Info: 973 (2), 163 | 2024-09
   - Links:   (optional) markdown links rendered as buttons, e.g.
              [arXiv](https://arxiv.org/abs/2509.18575) [Code](https://...)
   - Year:    The year this paper is grouped under in the timeline. EVERY paper
              below has one filled in, so YOU control the grouping — just edit
              the number to move a paper to a different year. (If you ever
              delete this line, the year falls back to auto-detection from the
              arXiv id / a YYYY-MM date in Info, then the venue year.)
   - URL:     (optional) The paper's link — the TITLE becomes clickable. If you
              omit it and Info has an arXiv id, the arXiv abstract link is used
              automatically (arXiv:2509.18575 -> https://arxiv.org/abs/2509.18575).
              Set URL for non-arXiv papers, e.g.  URL: https://doi.org/10.1111/jocd.15742

 Papers are GROUPED INTO A TIMELINE BY YEAR (newest year first), using the
 Year field above. Within a year they appear in the order written here. So you
 only need to keep papers of the same year in your preferred order; the years
 sort themselves.

 Tip: you can also make any text a link with [text](url). Lines outside a
 "## " block (like this comment) are ignored.
